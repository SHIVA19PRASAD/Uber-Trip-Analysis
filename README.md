Project Overview :
Developed an interactive Uber Trip Analysis Dashboard in Power BI to analyze booking trends, revenue generation, trip efficiency, and customer travel behavior for the period of June 2024. The dashboard provides a complete view of Uber trip performance using key operational and financial KPIs, helping businesses make better pricing, service, and customer experience decisions.

The project uses two tables: Uber Trip Details and Location Table, containing trip information such as pickup/drop-off time, passenger count, trip distance, fare amount, surge fee, vehicle type, payment method, and location mapping.
